Description:

For our unessay we have created a series of interactive puzzles that incorporate a range of security concepts designed to challenge the knowledge of the users on these topics. The goal of these puzzles is to let the users test their knowledge of these concepts in a safe and secure way. For each puzzle there is a code in each one at the end, when it is found it is to be input into the text box below the puzzle. If the code is correct the box will turn red, your goal is to get all three codes. The concepts here mainly revolve around web safety and security but there are elements of concepts like hashing, password security, sql injections and many more. By completing these puzzles, users will be able to better understand these concepts.
